Use this skill when the user needs a focused, repeatable workflow.

Workflow:
1. Identify the input and expected output.
2. Read only the context needed for this task.
3. Follow repository and project rules.
4. Produce the requested artifact or analysis.
5. Verify the result when verification is possible.
